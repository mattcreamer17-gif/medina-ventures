# Medina Ventures — Landing Page

A single-file static page prepared for Medina Ventures ahead of eMerge Americas 2026. The visual language mirrors medinavc.com: deep navy ground, warm off-white surfaces, a single muted gold accent, Playfair Display serif headlines over a neutral sans body. No Eduba branding above the footer. The only CTA is a 30-minute conversation with Matt Creamer via Calendly.

Intended URL path: /for/medina-ventures

## Files

- `index.html` — the page
- `styles.css` — brand tokens, layout, motion, and responsive rules (down to 375px)
- `logo.png` — Medina Ventures wordmark, served from the top bar
